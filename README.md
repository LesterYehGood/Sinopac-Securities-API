### Sinopac Securities API 下單


### Why

* 協助朋友自動排程國內期貨下單、偵測訂單做相對應回應

### How

* ASP.NET + AJAX 

### Program Language

* C#、Javascript

### Todolist

* 元件初始化
* 登入帳戶
* 憑證驗證測試
* 國內期貨下單
* 登出

### 功能

+ 手動下單
+ 排程下單
+ 策略預訂單
